Baislicka

About:
  Basic chess engine in C

Description:
  Using bitboards
  Iterative deepening depth-first search
  UCI Protocol
  Single threaded
  All times measured in ms
  Performance testing available
  Alpha-beta search

Platforms:
  Windows
  Linux
